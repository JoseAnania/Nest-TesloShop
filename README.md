# Ejecutar en Desarrollo

1. Clonar el repositorio
2. Ejecutar
```
npm install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Clonar el archivo ```.env.template```y renombrarlo a ```.env```
5. Cambiar las variables de entorno
6. Levantar la BD
```
docker-compose up -d
```
7. Levantar el modo desarrollo
```
npm run start:dev
```
